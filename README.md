# 📝 Test RR.HH.

App de estudio tipo certamen para **Gestión de Personas y Comportamiento Organizacional**.

Incluye 3 cuestionarios seleccionables:

- 📘 **Test RR.HH.** — 60 preguntas.
- 📗 **Guía de Estudio del Certamen** — 46 preguntas.
- 📕 **Guía — Nivel Avanzado** — 35 preguntas (estilo aplicado, mayor dificultad).

## Ejecutar localmente

```bash
pip install -r requirements.txt
streamlit run test_rrhh.py
```

Se abre en el navegador en `http://localhost:8501`.

## Hecho con

[Streamlit](https://streamlit.io).
